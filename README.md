# CodeAlpha_Iris_Classification
# Iris Flower Classification 🌸

## Project Overview
This project is part of the Data Science Internship at CodeAlpha.
The goal of this project is to build a Machine Learning model that classifies iris flowers into three species based on their physical measurements.

## Dataset
The dataset contains measurements of iris flowers:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Species classified:
- Setosa
- Versicolor
- Virginica

Dataset Source:
https://www.kaggle.com/datasets/saurabh00007/iriscsv

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Model
K-Nearest Neighbors (KNN) classification algorithm was used to train the model.

## Project Steps
1. Data Loading
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Model Evaluation

## Result
The model successfully classifies iris flower species with good accuracy.
